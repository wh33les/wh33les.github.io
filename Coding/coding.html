$\LaTeX$

$\LaTeX$ is a powerful typesetting language widely used for rendering complex math equations and formatting academic papers, including those for AMS (American Mathematical Society) publication.

I first learned $\LaTeX$ during my junior or senior year of undergraduate studies. I started by downloading a complete suite including MikTeX, which is the distribution and package manager, along with TeXnicCenter, a text editor with a compiler and a PDF preview feature. TeXnicCenter's built-in buttons for common LaTeX commands were particularly helpful for beginners.

As I became more proficient with $\LaTeX$, I transitioned away from TeXnicCenter and began compiling documents directly from Notepad++ using MikTeX, and rendering PDFs with Sumatra PDF, a lightweight viewer.

$\LaTeX$ has evolved over time, though some people still use its older incarnation, TeX, now called PlainTeX. My Ph.D. advisor used PlainTeX for his lecture notes, so I aimed to replicate his style for my own grad school presentations. Here is an example of a talk I created using PlainTeX.

In addition to typesetting, $\LaTeX$ supports creating posters and slide presentations. For instance, here is a poster that my students and I created. $\LaTeX$’s Beamer document class is specifically designed for making presentations, allowing extensive customization of color schemes and layouts. Here is an example of group theory lectures I prepared in summer 2016.

Lastly, $\LaTeX$ includes the TikZ package for creating graphics. While it has a learning curve, I invested time in mastering it during grad school. This talk and the poster linked above showcase some of the graphics I've made with TikZ.

Macaulay2

Macaulay2 (M2) is an open-source computer algebra system designed to work with Gröbner bases in its algorithms. It is available for download on FreeBSD, Linux, and Mac OS operating systems, but it does not support Windows natively.

During my time in grad school, I used M2 with instructions for running it through Emacs via a Linux emulator. Here is an example of a script I wrote in Emacs, including the outputs.

With advancements in technology, Windows Subsystem for Linux now allows you to run M2 directly on Windows. I typically use M2 from the command line, but many users still prefer running it through Emacs.
