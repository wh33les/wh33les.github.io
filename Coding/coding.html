Coding

This page just gives an overview of the languages I've used, and my history with learning them. You can find more information on my blog about learning code.

I've always liked coding, but for some reason I've never been able to become sufficiently proficient in any language. Of course, as a mathematician I've known how to use LaTeX since I was an undergrad, and since grad school I have used Macaulay2 in my research. But I've never made packages or templates with LaTeX, and I've only ever written small scripts in M2.

Recently I've started to pick up some code, though, namely Python and JavaScript, through the Erdös Institute's data science boot camp and data visualization minicourse. You can find the code I've written for their projects and problem sets on my Github account.

I actually started the blog in summer of 2017 when my contract at University of Arkansas ended and I was trying to learn basic computer science (algorithms, data structures) and implement what I was learning in C. However, I lost my momentum when I got hired at James Madison University. I picked it back up this past spring (2023) and started writing about what I've been learning from Erdös Institute. I even have a few posts about machine learning algorithms!

LaTeX

LaTeX is a typesetting language used to render math equations. It is the standard for formatting math papers for publication (AMS has several packages).

I first learned LaTeX in my junior or senior year of undergrad. To use it, I downloaded an entire suite with MikTeX, which is the distribution, its package manager, and a program called TeXnicCenter, a text editor with a compiler and a preview of the pdf rendered. What I liked about TeXnicCenter was that it had buttons for common LaTeX commands, and this is how I first learned them.

However, as I got more familiar with LaTeX, I found that I no longer needed TeXnicCenter. At one point during grad school, I found a way to write a script that would let me compile directly from Notepad++ (with MikTeX installed), then render the pdf in the lightweight pdf viewer, Sumatra PDF. I could try to remember how I did it, I still have the scripts I used, but for now I just use the suite with TeXworks.

LaTeX has a few notable incarnations. An earlier version exists, called TeX back when it was first implemented, and called PlainTeX now. Until recent years, my Ph. D. advisor wrote all of his lecture notes using PlainTeX, so when I was giving talks at the student seminars in grad school, I wanted to make my lecture notes look like his, so I learned it.
