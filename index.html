Learning React

I am currently (Feb 2025) working on a longer term project, an API that is a video game database for Husband. I wanted to use React for the front end, so this mini-project is just me experimenting around learning it. The apps on this page go in increasing difficulty.

(1) Tic tac toe tutorial

This tic tac toe game is based on the React Tic Tac Toe Tutorial, found here. At the end of the page on the React site, there is a list of suggestions for how to improve the game. Here are the suggestions, followed by a rendering of them all:

  1. For the current move only, show “You are at move #…” instead of a button.
  2. Rewrite Board to use two loops to make the squares instead of hardcoding them.
  3. Add a toggle button that lets you sort the moves in either ascending or descending order.
  4. When someone wins, highlight the three squares that caused the win (and when no one wins, display a message about the result being a draw).
  5. Display the location for each move in the format (row, col) in the move history list.
