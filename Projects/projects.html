Projects

  • Media bias vs. reliability

    This visualization plots the political bias (left/right) of around 800 news media sources against their reliability ratings, according to Ad Fontes Media. The data was scraped using Python and plotted using Tableau.

  • 2024 U.S. presidential election

    For my debut Tableau project, I analyzed over 4,000 polls spanning three years of data from FiveThirtyEight. Using Pandas, I applied the .groupby and .bfill methods to calculate daily rolling polling averages for candidates Donald Trump, Kamala Harris, and Joe Biden, covering the period from April 7, 2021, to August 11, 2024 (periodic updates since then). The Tableau dashboard allows users to toggle between polling data for likely voters, registered voters, and all voters.

  • "Do-nothing" Congress

    Using data from over 15,000 bills introduced in the 118th Congress, this project trains a model to predict whether a bill will become law.

  • Ashley's Fitbit stats

    This project showcases a selection of my personal fitness data using newly learned d3.js. Check out the full project in my GitHub repository.

  • 538 project

    I wrote a script to scrape metadata from FiveThirtyEight (538). A key feature of this script is a custom function that renders JavaScript on each page to extract the number of comments from the Facebook plugin.
